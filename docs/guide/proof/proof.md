::: note Introduction
This document aims to help authors quickly understand the core functions of the OEP Proofreading Center, master various operational processes, and achieve efficient proofreading collaboration.
:::

## How to log in

- Click the exclusive <span style="color:#87CEFA">URL</span> link sent by the Assistant Editor to access the platform's login page.
- **Login Credentials:** Log in using your ARES account.
- **Account Registration:** If you do not have an ARES account, click the <span style="color:#87CEFA">Create an account</span> button on the login page, complete the new account registration as guided, and then return to log in.

## How to complete the process

:::: steps
1. **Step 1: Download**

   Download all files requiring proofreading.

2. **Step 2: Proofread**

   - Address all comments added by the Editorial Office in the manuscript one by one to ensure no omissions.
   - Carefully read and sign the publication license file.

3. **Step 3: Upload**

   Upload the revised manuscript and the signed publication license file.

4. **Step 4: Submit**

   Click the <Badge type="tip" text="Submit" /> button to finalize the submission.

::::

## How to download

The platform supports two download options, with bulk download recommended:

- **Bulk Download:** Click the <Badge type="tip" text="Download" /> button to download all files requiring proofreading at once (usually including the main manuscript document, publication license file, supplementary materials, etc.).
- **Single Download:** Click the <span style="color:#87CEFA">::charm:download::</span> button next to the corresponding file to download only the specified file.

## How to upload

Two operation options are supported: click-to-upload and drag-and-drop upload. The complete upload process consists of two steps:

::: demo-wrapper img no-padding
![Upload_Animation](/guide/proof/fig/Upload_Animation.gif)
:::

:::: steps
1. **Step 1: File Upload**

   Prepare all files to be submitted (including the revised manuscript, signed publication license, etc.) and upload them by clicking the "<u style="color:#87CEFA">Link</u>" or using drag-and-drop.

   ::: note
   If you inadvertently upload an incorrect file, click the <span style="color:red">::material-symbols:delete::</span> icon next to the file to delete the corresponding incorrect file.
   :::

2. **Step 2: Confirm Submission**

   Click the <Badge type="tip" text="Submit" /> button to complete the submission.

   ::: warning Reminder
   Please make sure to click Submit; otherwise, the upload will not be completed successfully.
   :::

::::

## How to delete files

Files can be deleted at any time after submission. Specific operations:

- **Bulk Delete:** Click the <span style="color:#87CEFA">Delete</span> button in the file list area to delete all files uploaded this time.
- **Single Delete:** Click the <span style="color:red">::material-symbols:delete::</span> icon next to the target file to delete that file individually.

## How to use the comment function

If you need to provide additional explanations to the Editorial Office, you can use the comment function. This function supports replying to, deleting, and modifying comments that have already been saved.

:::: steps
1. **Step 1: Open the Comment Box**

   In each file upload area, click the <span style="color:#87CEFA">::mdi:comment-processing::</span> button to pop up the comment input box.

2. **Step 2: Edit Comments** 

   Freely edit text content in the input box. Text formatting (including Markdown format) and image upload are not supported.

3. **Step 3: Save Comment**

   Click the <Badge type="tip" text="Comment" /> button to save your comment.

::::

::: tip Tips
Each comment has a maximum limit of 1000 characters.
:::
